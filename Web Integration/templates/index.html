Visualization Tool for Electric Vehicle Charge and Range Analysis

A structured and interactive analytics platform designed to provide insights into EV charging infrastructure, performance metrics, powertrain distribution, and pricing trends.

About the Project

This project integrates SQL-based data preprocessing and Tableau interactive visualization to analyze electric vehicle market data.

It provides structured insights into charging station distribution, brand comparison, efficiency analysis, powertrain segmentation, and EV pricing trends across India.

The platform eliminates scattered data sources and presents everything in a unified and visually intuitive dashboard environment.
